## Performance Characteristics

### Latency Breakdown (typical query)
```

Vector Search:          0.05-0.1s (local FAISS)
Reranking:             0.2-0.5s  (cross-encoder)
Response Generation:    1-3s     (LLM call)
Evaluation:            <0.01s   (heuristic)

Total:                 2-5s
```

### Throughput
- **Local embeddings**: Unlimited (no API quota)
- **Vector search**: ~1000 queries/sec
- **Reranking**: ~10-50 queries/sec (depends on hardware)
- **LLM generation**: Limited by API rate limits

## Scalability Points

### Current Setup (Single Instance)
- Handles 1-10 concurrent users
- In-memory conversation state
- Local FAISS index

### Production Recommendations
- Use Redis for conversation memory
- Deploy vectorstore to vector DB (Pinecone, Weaviate)
- Add caching layer for common queries
- Implement request queuing
- Scale backend horizontally
- Add load balancer
